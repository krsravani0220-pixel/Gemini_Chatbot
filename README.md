\# 🤖 Gemini AI Assistant



A modern AI-powered conversational assistant built using \*\*Google Gemini 2.5 Flash\*\* and \*\*Streamlit\*\*.



The application features conversation memory, Google Search Grounding, and a clean SaaS-inspired interface.



\---



\## ✨ Features



\- 💬 Real-time AI Conversations

\- 🧠 Conversation Memory

\- 🌐 Google Search Grounding

\- 🎨 Modern UI

\- ⚡ Fast Responses

\- 📱 Responsive Design



\---



\## 🛠️ Tech Stack



\- Python

\- Streamlit

\- Google GenAI SDK

\- Gemini 2.5 Flash

\- Python Dotenv



\---



\## 🚀 Installation



Clone the repository



```bash

git clone https://github.com/krsravani0220-pixel/Gemini\_Chatbot.git

```



Move into the project



```bash

cd Gemini\_Chatbot

```



Install dependencies



```bash

pip install -r requirements.txt

```



Create a `.env` file



```text

GEMINI\_API\_KEY=YOUR\_API\_KEY

```



Run the application



```bash

streamlit run app.py

```
